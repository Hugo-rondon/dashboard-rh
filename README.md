# Dashboard RH

Versão web estática do Dashboard RH, preparada para publicação no GitHub Pages.

## Atualização mensal

1. Substitua ou atualize as planilhas na pasta de origem.
2. Execute `atualizar-dashboard.cmd` na pasta principal do projeto.
3. Confira o arquivo `dashboard-rh/index.html`.
4. Envie a pasta `dashboard-rh` ao repositório no GitHub.

O arquivo publicado contém somente dados consolidados. Nomes e matrículas não são enviados ao site.

## GitHub Pages

No repositório, abra **Settings > Pages**, selecione **Deploy from a branch**, escolha a branch principal e a pasta raiz. O endereço será disponibilizado pelo GitHub após a publicação.
